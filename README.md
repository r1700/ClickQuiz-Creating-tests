# 📝 ClickQuize - Teacher Test Creator

**An interactive web application that allows teachers to quickly create, manage, and share exams, with a strong focus on using AI for automatic question generation.**

---

## 🌟 About the Project
In today's world, teachers face a heavy workload. **Teacher Test Creator** provides a smart and innovative solution: generating customized exams while leveraging **AI technology** to automatically create questions and answers.  

The system allows a mix of **manual questions** and **AI-generated questions**, streamlining the exam preparation process and enhancing creativity.  

### Key benefits of AI integration:
- **Time-saving** – no need to write every question manually  
- **Personalization** – create questions based on subject, grade, level, and topic  
- **Variety & creativity** – generate unique and diverse questions for any topic  
- **Enhanced quality** – automatically add explanations for correct answers  

---

## 🎯 Project Goals
- Make exam creation faster, smarter, and more accessible  
- Leverage AI to automatically generate questions and answers  
- Allow different exam versions: teacher version with answers, student version without answers  
- Enable easy saving, editing, and sharing of exams  

---

## 👩‍🏫 Target Users
- Teachers at all educational levels: elementary, middle, and high school  
- Teaching teams who want to create and distribute exams efficiently  

---

## ⚙️ Key Features
### 🆕 AI-Powered Exam Creation
- Automatic question and answer generation using AI  
- Option to combine manual and AI-generated questions  
- Add explanations for correct answers  
- Set basic parameters: subject, topic, grade, class, number of questions, question types  

### 🗂️ Exam Management
- Full CRUD: create, read, update, delete  
- Store exams in a personal MongoDB database  
- Search and filter by topic, class, difficulty level, and date  

### 📤 Sharing & Distribution
- Versions for teachers (with answers) and students (without answers)
- Download as PDF or Word  
- Share via link, QR code, or email  
- Direct printing from the system  

### 👥 User System
- Registration and login  
- Personal profile with created exams and sharing options  
- Permissions: teachers can only edit their own exams  

---

## 💻 Technologies
- **Frontend:** React, Material-UI  
- **Backend:** Node.js, Express  
- **Database:** MongoDB  
- **AI:** OpenAI API
